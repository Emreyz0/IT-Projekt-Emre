# IT-Projekt-Emre
Kleines eigenes Projekt mit Python (Flask).

## Funktionen
- Benutzerverwaltung
- Geräteübersicht
- Einfache Statistik

## Starten der App (lokal)
1. Python 3.x installieren  
   https://www.python.org/downloads/

2. Projekt entpacken

3. CMD / Terminal öffnen und in den Projektordner wechseln:
   cd Pfad\zum\Projektordner

4. Flask installieren:
   pip install flask

5. App starten:
   python app.py

6. Browser öffnen:
   http://127.0.0.1:5000
